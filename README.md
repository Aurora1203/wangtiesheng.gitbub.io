# 王铁生简介

### 王铁生,男,纳西族,1955年出生于云南丽江古城,中国音乐著作权协会会员，云南省音乐家协会会员,丽江市群众艺术馆馆员。自小深受当地文化界琴艺名师和云池及父亲王钦文的栽培,到高中毕业时,能基本正确掌握小提琴演奏,并能熟用多种记谱法，73年响应中央号召,到农村当知识青年…

      三中全会以后,78年考入丽江地区歌舞团,从此走上了专业音乐工作道路。曾多次参加国家省民族文艺调研活动、《聂耳音乐周》活动、国家艺术节活动,多年努力完成国家指不定期的职业演出工作及下乡任务,取得优异成绩。曾在长春音乐家协会作曲专业班学习作曲。85年调

*至丽江地区文化局创作室工作,86年调至丽江地区群众艺术馆音乐部工作,多年收集整理纳西等民族音乐,先后创作纳西族东巴舞蹈音乐《灯花舞》、普米族舞蹈音乐《打麦歌》、摩梭族舞蹈音乐《成人礼》等音乐。并在省地区获奖。长期在艺术馆担任音乐编辑、音乐教学、文艺辅导等工作。*

> 曾完成国家指定的工作项目,即《民族器乐集成》,民族民间艺人调查,文化普査等。长期以来为民族音乐的发掘和整理走遍云岭山谷。长期深入各民族的生活,与当地的纳西族、白族、彝族、傈僳族、普米族、藏族等12个少数民族的同胞结下了深厚情义与不解之缘。为音乐创作打下了牢靠的基础。

## *创作《美丽的泸沽湖》传唱不久,于97年丽江作品《丽江行》,此歌以丽江县政府、由著名歌唱家阎维文演唱后,与丽江山水风情的介绍制作成第一个MTV,向全国发行,多次在中央电视台播放后,在各地不断传唱,得到有关领导的肯定和群众的好评。*

    此后,云南十八怪为歌词创作了可以传唱的第一首歌曲《云南十八怪》。2002年作品专集《月下古城》出版发行:2003年至2006年间陆续创作出《西部丽江1.)、《西郭丽江2.》。在作品《从月亮走向太阳》，《白乡靓姿》，《傈僳酒歌》，《河灯情思》、《文笔神山》等的基础上仍在坚贞不渝的从事音乐创作，,把自己的创力进一步深化,力求出更好的作品,培育更优秀的演奏人员,将更好流动的音乐建筑凝固入社会主义文化的高楼大厦。
